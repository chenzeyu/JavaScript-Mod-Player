Welcome to JavaScript Mod Player.

As the name indicates, this is a player of mod music format. Please try out our demo!
Chinese Version

User Guide

  • You can choose your own mod files on your computer to play, or choose to play the demos we provided.
  • Don't click another song when one is playing.
  • No playback control provided, you can refresh the page to stop the song.

Limitations

  • Only several effects are implemented
  • Consume a lot of system resources
  • No playback control provided
  • Implemented by using web audio api, therefore IE is not supported
